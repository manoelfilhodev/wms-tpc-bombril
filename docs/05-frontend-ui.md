# Frontend e UI Systex

Este documento define a base visual e operacional para telas administrativas em Blade.

## Direcao visual

- Visual SaaS profissional.
- Tema dark/glass.
- Base em preto, cinza e branco.
- Vermelho apenas como destaque.
- Cards escuros com bordas sutis.
- Botoes modernos e estados visuais claros.
- Interface limpa, tecnologica e operacional.

## Principios de interface

- Priorizar leitura rapida, densidade organizada e fluxo operacional.
- Evitar excesso de elementos decorativos.
- Manter formularios previsiveis, com mensagens de erro claras.
- Usar componentes reutilizaveis quando houver padrao estabelecido.
- Garantir responsividade para uso administrativo e possivel operacao em dispositivos moveis.

## Cuidados

- Nao alterar layout global sem avaliar impacto em telas existentes.
- Nao remover informacoes operacionais usadas por equipes de deposito, conferencia ou gestao.
- Preservar acessibilidade basica: contraste, foco, rotulos e feedback de estado.
